MapleStory Client - UI Theme System

Project Requirements:
The goal is to implement a customizable UI theme system for the MapleStory client that allows players to personalize their gaming experience.

Features:
1. Theme Selection - Players can choose from predefined themes (dark, light, classic)
2. Color Customization - Allow custom color selection for UI elements
3. Transparency Control - Adjustable transparency for windows and panels
4. Theme Persistence - Save and load theme settings
5. Real-time Preview - See changes immediately without restart

Technical Requirements:
- Integrate with existing C++ UI framework
- Store theme settings in configuration file
- Ensure performance doesn't degrade with themes
- Support both keyboard and mouse interaction for theme selection

Priority: High
Estimated Effort: 3-4 weeks
